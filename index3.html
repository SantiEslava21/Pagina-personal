Artículos sobre el desarrollo de JustMe

Reflexiones, aprendizajes y experiencias durante la creación del proyecto JustMe.

1. Cómo nació JustMe: Conectando belleza y tecnología

JustMe surgió como un proyecto académico del SENA enfocado en unir a profesionales del sector belleza con clientes que buscan servicios a domicilio. Durante su desarrollo, aprendimos a trabajar con metodologías ágiles, a estructurar interfaces funcionales y a aplicar buenas prácticas de diseño.

La idea principal fue crear una aplicación accesible, visualmente atractiva y que solucionara una necesidad real: ofrecer servicios de belleza personalizados de forma rápida y segura.

2. Tecnologías utilizadas en JustMe

El proyecto combina herramientas del frontend y backend para lograr una plataforma completa:

  • HTML5 y CSS3: estructura y diseño visual del sitio.
  • JavaScript: funciones interactivas y formularios dinámicos.
  • Java (Spring Boot): lógica del sistema y controladores.
  • MySQL: base de datos para usuarios, servicios y reservas.
  • GitHub: control de versiones y trabajo colaborativo.

3. Retos y aprendizajes durante el desarrollo

Uno de los mayores retos fue integrar correctamente el backend en Java con el frontend en HTML, CSS y JavaScript. También aprendimos a manejar la comunicación entre las vistas de usuario y los datos almacenados en MySQL, así como a depurar errores en Spring Boot.

Este proyecto fortaleció nuestras habilidades como desarrolladores Full Stack, y reforzó la importancia de la documentación, el trabajo colaborativo y las buenas prácticas en el desarrollo web.
