JustMe (SENA)

Descripción: Plataforma web que conecta profesionales del área de belleza con clientes que buscan servicios a domicilio.

Lenguajes y Tecnologías:

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Java (Spring Boot)
  • Base de datos: MySQL
  • Control de versiones: Git y GitHub

Objetivo: Facilitar la contratación de servicios de belleza a domicilio, mejorando la visibilidad de los profesionales y la experiencia de los usuarios.
